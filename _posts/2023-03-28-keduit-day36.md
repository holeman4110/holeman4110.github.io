3. EIGRP
//R1
# router eigrp 28

//R2
# router eigrp 28

# network 28.28.2.2 0.0.0.0

# network 28.28.24.2 0.0.0.0
# no auto

```
//Switching Types
1. Process Switching: 출발지 부하분산
# no ip route-cache //Process Switching. Under Interface + no ip cef
2. Fast Switching(Route Caching): 목적지 부하분산, 대부분 Cisco Router의 Default
# ip route-cache //Fast Switching. Under Interface
3. CEF(Cisco Express Forwarding): 출발지 목적지별 부하분산
# ip cef //CEF. Under Interface
```

```
Unequal Cost Loadbalancing
1. [Best Path's FD] > Backup Path's RD
2. [Backip Path's Metric] + Variance > [Best Path's FD]
```

```
//Feasible Successor를 만든 후 Unequal Loadbalancing 만들기
//R1
# int s1/0.13
# delay 3000
//R3
# int s1/0.13
# delay 3000
//R1
# router eigrp 28
# variance 2
```
